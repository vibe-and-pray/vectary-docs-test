# Text



The **Text** UI element displays text within the UI and supports custom fonts.

Text can also be dynamic by using **expressions** and **variables -** [variables-and-expressions](../variables-and-expressions/ "mention")



* **Custom fonts** — Custom text can be imported in **`OTF`** format.
* **Text settings** — Includes all standard text formatting options similar to those found in text editors, such as font selection, size, weight, alignment, and color.

# Download image



The **Download Image** action captures and saves a snapshot of either the current view or a specified camera view from the selected camera within the project, allowing end users to export high-quality images.



<figure><img src="../../../../.gitbook/assets/image (225).png" alt="" width="563"><figcaption></figcaption></figure>

**Settings**

* **Camera** — Select the camera from which the screenshot will be taken.
* **Custom resolution** — Set a specific resolution for the screenshot.
* **Transparent background** — Enable a transparent background for the screenshot.

{% hint style="info" %}
**Custom resolution** — only one resolution parameter (width or height) can be manually set, while the other will be automatically adjusted to preserve the aspect ratio upon download. For instance, if the default resolution is **Width: 2464** and **Height: 1898**, and Width is changed to **1000**, the final downloaded image resolution will be **1000 × 770**.
{% endhint %}

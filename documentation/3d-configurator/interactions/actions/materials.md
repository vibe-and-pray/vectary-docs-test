# Materials



The **Materials** action allows switching between existing materials assigned to objects.

<figure><img src="../../../../.gitbook/assets/image (228).png" alt="" width="563"><figcaption></figcaption></figure>



**Settings**

* The target can be either a specific object or [selections.md](../../../other/selections.md "mention").
* **Material:**
  * **Next material** — Switches to the next material assigned to the selected object.
  * **Previous material** — Switches to the previous material assigned to the selected object.
  * **Previously used** — Restores the last active material.
  * _"Material name"_ — Select a specific material by name to be applied to the object.



{% hint style="info" %}
Material switching can also be set up through [materials-ui.md](../../floating-ui/materials-ui.md "mention") (Floating UI element)
{% endhint %}

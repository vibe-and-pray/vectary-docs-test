# Import tips



## Replace geometry



The context menu of [**mesh objects**](#user-content-fn-1)[^1] includes the **Replace geometry** option. This action allows importing a new object in place of the current one while preserving its position, scale, and rotation.

Supported formats for this type of import: `FBX`, `GLTF`, `GLB`, `OBJ`, `STL`

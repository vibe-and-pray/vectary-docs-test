# Comments



Comment Mode enables placing comments directly within the 3D scene, facilitating collaboration and feedback. This mode supports efficient communication within the workspace and helps streamline project reviews and iterations.



#### Key features



* Comments can be added directly to any location in the scene
* Workspace members can be mentioned using **`@`** followed by their name
* Mentioned users receive an email notification if email alerts are enabled in their account settings\
  <img src="../../.gitbook/assets/image (458).png" alt="" data-size="original">
* Comments can also serve as personal notes or reminders within the project

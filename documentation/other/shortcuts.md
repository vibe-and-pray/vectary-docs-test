# Shortcuts



The Shortcuts page provides a comprehensive list of keyboard shortcuts designed to speed up the workflow in Vectary Studio. These shortcuts cover navigation, object manipulation, editing tools, and more.

{% hint style="success" %}
Note: Wherever **`Ctrl`** is mentioned, use **`Cmd`** on macOS
{% endhint %}

# Offline Viewer

{% hint style="info" %}
Available only on the Business plan.
{% endhint %}



Vectary Offline Viewer enables the offline presentation and demonstration of Vectary projects in environments with limited or no internet connectivity.

The viewer is based on a **Progressive Web App (PWA)** that runs on **Windows**, **macOS**, and touch devices supporting split-screen view, including **iPads** and **Android** tablets.



## Installation instructions



{% stepper %}
{% step %}
Go to Vectary Studio and open **Menu** → **Export** → select **`Vectary`** option in **File format**

<div align="left"><figure><img src="../../.gitbook/assets/image (20).png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Click on **`Get Offline viewer`**

<div align="left"><figure><img src="../../.gitbook/assets/image (21).png" alt="" width="563"><figcaption></figcaption></figure></div>

This will open a download link in a new tab (follow the platform-specific instructions):

{% tabs %}
{% tab title="Chrome (Mac/PC)" %}


Click the download icon on the right side of the address bar and install the app.\
\
![](<../../.gitbook/assets/image (19).png>)
{% endtab %}

{% tab title="Safari (Mac)" %}


Click the Share button and select **`Add to Dock`**.

<div align="left"><figure><img src="../../.gitbook/assets/image (18).png" alt="" width="317"><figcaption></figcaption></figure></div>
{% endtab %}

{% tab title="Safari (iPad)" %}


Click the **`Share`** button and select **`Add to Home Screen`**.
{% endtab %}

{% tab title="Chrome (Android)" %}


Tap the three-dot icon and select **`Add to Home Screen`**.
{% endtab %}
{% endtabs %}
{% endstep %}
{% endstepper %}



## Loading a Vectary project



Once the app is installed, download the final version of the Vectary project:



{% stepper %}
{% step %}
Go to Vectary Studio and open **Menu** → **Export** → select `Vectary` option in **File format**


{% endstep %}

{% step %}
Click on **`Download project`**

The file will be saved in **`.vctr3`** format.

<div align="left"><figure><img src="../../.gitbook/assets/image (22).png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Open the Offline Viewer and drag and drop the **`.vctr3`** file into the blank white canvas.

{% hint style="info" %}
For touch devices, a split-screen view with the Files app is recommended for easier drag-and-drop interaction.
{% endhint %}
{% endstep %}
{% endstepper %}



## Notes



{% hint style="danger" %}
#### Keep the Viewer app up to Date

Always download the latest version of the app when downloading the project file to ensure compatibility with the most recent Vectary features and updates.
{% endhint %}



{% hint style="warning" %}
**AR support is currently unavailable**. Integration is under development.
{% endhint %}
